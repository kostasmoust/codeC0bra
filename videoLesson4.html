Loops In Python

Επαναλήψεις: Οι επαναλήψεις είναι ένας τρόπος να λες στην Python: «Κάνε κάτι ξανά και ξανά, μέχρι να σου πω να σταματήσεις.» Είναι βασικές για προγράμματα που χρειάζονται να εκτελέσουν την ίδια εντολή πολλές φορές.

Υπάρχουν δύο βασικοί τύποι επαναλήψεων στην Python: for και while.

1. Επανάληψη for – Για καθορισμένο αριθμό φορών:

Χρησιμοποιείται όταν ξέρουμε από πριν πόσες φορές θα γίνει η επανάληψη.

for i in range(5):
    print(i)

for name in ["Γιάννης", "Άννα", "Κώστας"]:
    print(name)

2. Επανάληψη while – Όσο ισχύει μια συνθήκη:

Χρησιμοποιείται όταν ΔΕΝ ξέρουμε από πριν πόσες φορές θα επαναληφθεί κάτι.

x = 0
while x < 5:
    print(x)
    x += 1

3. Χρήση του break – Διακοπή της επανάληψης:

Σταματά την επανάληψη όταν συμβεί κάτι συγκεκριμένο.

for i in range(10):
    if i == 5:
        break
    print(i)

4. Χρήση του continue – Παράλειψη της τρέχουσας επανάληψης:

Αγνοεί την υπόλοιπη επανάληψη και συνεχίζει με την επόμενη.

for i in range(10):
    if i % 2 == 0:
        continue
    print(i)

Συμπέρασμα: Οι επαναλήψεις είναι απαραίτητες για την επεξεργασία λιστών, επαναλαμβανόμενων ενεργειών και δεδομένων. Χρησιμοποίησε for όταν ξέρεις πόσες φορές θα επαναλάβεις, και while όταν η επανάληψη εξαρτάται από μια συνθήκη.
