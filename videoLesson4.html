Επαναλήψεις Οι επαναλήψεις είναι ένας τρόπος να λες στην Python: «Κάνε κάτι ξανά και ξανά, μέχρι να σου πω να σταματήσεις.» Είναι απαραίτητες για κάθε πρόγραμμα που χρειάζεται να εκτελέσει την ίδια εντολή πολλές φορές. Υπάρχουν δύο βασικοί τύποι επαναλήψεων στην Python: for και while. 🔹 1. Επανάληψη for – Για καθορισμένο αριθμό φορών Χρησιμοποιείται όταν ξέρουμε πόσες φορές θέλουμε να επαναλάβουμε κάτι. (π.χ. τύπωσε τους αριθμούς από το 0 ως το 4: for i in range(5):   print(i)) (π.χ. για κάθε όνομα μέσα στη λίστα ονόματα, τύπωσε το όνομα: for name in ["Γιάννης", "Άννα", "Κώστας"]:   print(name)) 🔹 2. Επανάληψη while – Όσο ισχύει μια συνθήκη Χρησιμοποιείται όταν δεν ξέρουμε από πριν πόσες φορές θα επαναληφθεί κάτι, αλλά βασιζόμαστε σε μια συνθήκη. (π.χ. όσο το x είναι μικρότερο από 5, τύπωσε το x και αύξησέ το κατά 1: x = 0 while x < 5:   print(x)   x += 1) 🔹 3. Χρήση του break – Σταματά την επανάληψη (π.χ. επανάλαβε για τους αριθμούς 0 ως 9, αλλά σταμάτα όταν φτάσεις στο 5: for i in range(10):   if i == 5:     break   print(i)) 🔹 4. Χρήση του continue – Παραλείπει την τρέχουσα επανάληψη (π.χ. τύπωσε μόνο τους περιττούς αριθμούς από 0 ως 9: for i in range(10):   if i % 2 == 0:     continue   print(i)) Συμπέρασμα Οι επαναλήψεις είναι απαραίτητες για να χειρίζεσαι λίστες, δεδομένα και επαναλαμβανόμενες διαδικασίες. Το for είναι ιδανικό όταν ξέρεις πόσες φορές θα επαναλάβεις κάτι, ενώ το while είναι καλύτερο όταν περιμένεις κάτι να συμβεί για να σταματήσεις.
