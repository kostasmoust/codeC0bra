Intermediate Expressions

Προχωρημένες Εκφράσεις:
Αφού μάθεις τις βασικές
προτάσεις και τελεστές στην Python, μπορείς να γράφεις πιο σύντομα και αποδοτικά με προχωρημένες εκφράσεις. Είναι απλές τεχνικές που κάνουν τον κώδικα καθαρότερο και πιο κομψό.

1. Τριπλές Εκφράσεις (Ternary Expressions): Σύντομος τρόπος για if-else.

status = "Ενήλικας" if age >= 18 else "Ανήλικος"

2. Εκφράσεις Λίστας (List Comprehensions): Σύντομος τρόπος δημιουργίας λιστών.

numbers = [i for i in range(10)]
squares = [i*i for i in range(10)]
evens = [i for i in range(10) if i % 2 == 0]

3. Εκφράσεις με any() και all(): Έλεγχος λίστας με μία γραμμή.

any(x > 10 for x in numbers)
all(x > 0 for x in numbers)

4. Εκφράσεις με zip() και enumerate(): Χρήσιμες για συνδυασμούς λιστών ή εύρεση index.

zip(["Γιάννης", "Άννα"], [23, 25]) → [("Γιάννης", 23), ("Άννα", 25)]
enumerate(["Μήλο", "Μπανάνα"]) → [(0, "Μήλο"), (1, "Μπανάνα")]

5. Εκφράσεις με lambda (Ανώνυμες Συναρτήσεις): Γρήγορες, σύντομες συναρτήσεις.

διπλασιο = lambda x: x * 2
διπλασιο(4) → 8

list(filter(lambda x: x % 2 == 0, [1, 2, 3, 4])) → [2, 4]

Συμπέρασμα: Οι προχωρημένες εκφράσεις σου επιτρέπουν να γράφεις πιο σύντομα, καθαρά και αποτελεσματικά. Δεν είναι δύσκολες — απλώς απαιτούν εξάσκηση για να εξοικειωθείς με τη σύνταξή τους.
