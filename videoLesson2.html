Προχωρημένες Εκφράσεις Αφού μάθεις τις βασικές προτάσεις και τελεστές στην Python, είναι ώρα να γνωρίσεις τις προχωρημένες εκφράσεις, δηλαδή τρόπους να γράφεις πιο σύντομο και «πιο έξυπνο» κώδικα. Αυτές οι εκφράσεις δεν είναι απαραίτητα προχωρημένες, αλλά βοηθούν πολύ στο να γίνει ο κώδικάς σου πιο καθαρός και αποτελεσματικός. 1. Τριπλές Εκφράσεις (Ternary Expressions) – Σύντομο if else Αντί να γράφεις πολλές γραμμές για να ελέγξεις μια συνθήκη, μπορείς να χρησιμοποιήσεις μια σύντομη έκφραση if else. (π.χ. δώσε την τιμή "Ενήλικας" αν η ηλικία είναι 18 ή παραπάνω, αλλιώς "Ανήλικος": status = "Ενήλικας" if age >= 18 else "Ανήλικος") 🔹 2. Εκφράσεις Λίστας (List Comprehensions) Πολύ σύντομος τρόπος για να δημιουργήσεις λίστες με βάση έναν κανόνα ή μια επανάληψη. (π.χ. δημιούργησε μια λίστα με τους αριθμούς από το 0 ως το 9: numbers = [i for i in range(10)]) (ή πάρε μόνο τους τετραγώνους αριθμούς από το 0 έως το 9: squares = [i*i for i in range(10)]) (ή φτιάξε λίστα με μόνο τους ζυγούς αριθμούς από 0 έως 9: evens = [i for i in range(10) if i % 2 == 0]) 🔹 3. Εκφράσεις με any() και all() Χρήσιμες για έλεγχο σε λίστες ή συλλογές. (π.χ. ελέγχει αν τουλάχιστον ένας αριθμός στη λίστα είναι μεγαλύτερος του 10: any(x > 10 for x in numbers)) (ή ελέγχει αν όλοι οι αριθμοί είναι θετικοί: all(x > 0 for x in numbers)) 🔹 4. Εκφράσεις με zip() και enumerate() Εργαλεία για να χειρίζεσαι πολλαπλές λίστες ή να παίρνεις και την τιμή και το index. (π.χ. για να ενώσεις δύο λίστες στοιχείο προς στοιχείο: zip(["Γιάννης", "Άννα"], [23, 25]) → [("Γιάννης", 23), ("Άννα", 25)]) (ή για να πάρεις index και στοιχείο από μια λίστα: enumerate(["Μήλο", "Μπανάνα"]) → [(0, "Μήλο"), (1, "Μπανάνα")]) 🔹 5. Εκφράσεις με lambda (Ανώνυμες Συναρτήσεις) Μικρές συναρτήσεις μιας γραμμής χωρίς όνομα, χρήσιμες για σύντομες μετατροπές. (π.χ. συνάρτηση που επιστρέφει το διπλάσιο ενός αριθμού: διπλασιο = lambda x: x * 2 διπλασιο(4) → 8) (ή φίλτρο που κρατά μόνο τους ζυγούς αριθμούς από μια λίστα: list(filter(lambda x: x % 2 == 0, [1, 2, 3, 4])) → [2, 4]) Συμπέρασμα Οι προχωρημένες εκφράσεις είναι το επόμενο βήμα για να γράφεις Python με μεγαλύτερη ευελιξία και κομψότητα. Δεν είναι δύσκολες — απλώς χρειάζονται εξάσκηση για να συνηθίσεις τη σύνταξη και τη λογική τους.
