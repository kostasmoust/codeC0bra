Συνθήκη Εκτέλεσης Η συνθήκη εκτέλεσης είναι το θεμέλιο της λήψης αποφάσεων σε ένα πρόγραμμα. Στην Python, μπορείς να γράψεις εντολές που εκτελούνται μόνο αν πληρούνται κάποιες προϋποθέσεις — με άλλα λόγια, αν μια συνθήκη είναι αληθής. 🔹 1. Η βασική δομή if Η απλούστερη μορφή συνθήκης. Αν μια συνθήκη είναι True, τότε εκτελείται το μπλοκ του if. (π.χ. αν το x είναι μεγαλύτερο από το 0, τύπωσε "Θετικός": if x > 0:   print("Θετικός")) 🔹 2. Η δομή if...else Εκτελεί μία εντολή αν η συνθήκη είναι True, και μια άλλη αν είναι False. (π.χ. αν το x είναι μεγαλύτερο από το 0, τύπωσε "Θετικός", αλλιώς τύπωσε "Μη θετικός": if x > 0:   print("Θετικός") else:   print("Μη θετικός")) 🔹 3. Η δομή if...elif...else Για πολλαπλές συνθήκες. Το πρόγραμμα εκτελεί το πρώτο True μπλοκ που βρίσκει. (π.χ. αν ο βαθμός είναι 90 ή παραπάνω, τύπωσε "Άριστα", αν είναι 70–89, τύπωσε "Καλά", αλλιώς τύπωσε "Ανεπαρκής": if grade >= 90:   print("Άριστα") elif grade >= 70:   print("Καλά") else:   print("Ανεπαρκής")) 🔹 4. Συνθήκες με and, or, not Μπορείς να συνδυάσεις πολλαπλές συνθήκες με λογικούς τελεστές. (π.χ. αν ο χρήστης είναι πάνω από 18 ετών και έχει ταυτότητα: if age > 18 and has_id:   print("Πρόσβαση επιτρεπτή")) (π.χ. αν η θερμοκρασία είναι κάτω από 0 ή βρέχει: if temperature < 0 or is_raining:   print("Πάρε ομπρέλα ή ντύσου καλά")) (π.χ. αν ΔΕΝ έχει κωδικό πρόσβασης: if not has_password:   print("Απαιτείται κωδικός")) 🔹 5. Σύντομη συνθήκη (Ternary Expression) Μπορείς να γράψεις μια if...else συνθήκη σε μία γραμμή. (π.χ. τύπωσε "Ενήλικας" αν η ηλικία είναι τουλάχιστον 18, αλλιώς "Ανήλικος": print("Ενήλικας") if age >= 18 else print("Ανήλικος")) ________________________________________ Προσοχή στην Εσοχή (Indentation) Η Python δεν χρησιμοποιεί αγκύλες όπως άλλες γλώσσες. Το ποιο μπλοκ ανήκει σε ποια συνθήκη καθορίζεται από την εσοχή (τα κενά στην αρχή της γραμμής). (π.χ. αν ξεχάσεις να βάλεις εσοχή στο print, η Python θα σου βγάλει λάθος: if x > 0: print("Θετικός") ← λάθος, πρέπει να έχει εσοχή) Συμπέρασμα Η συνθήκη if είναι το «αν» του προγραμματισμού — και με τους συνδυασμούς elif, else, and, or, και not, μπορείς να πεις στον υπολογιστή να παίρνει αποφάσεις όπως θα έκανες κι εσύ.
