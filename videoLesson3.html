Conditional Structures

Συνθήκη Εκτέλεσης: Οι συνθήκες είναι ο βασικός τρόπος για να παίρνει αποφάσεις ένα πρόγραμμα. Στην Python, εκτελούνται μόνο αν κάποια λογική συνθήκη είναι True.

1. Η βασική δομή if: Εκτελείται μόνο αν η συνθήκη είναι αληθής.

if x > 0:
    print("Θετικός")

2. Η δομή if...else: Αν η συνθήκη είναι True, εκτελείται το πρώτο μπλοκ. Αλλιώς, εκτελείται το else.

if x > 0:
    print("Θετικός")
else:
    print("Μη θετικός")

3. Η δομή if...elif...else: Επιτρέπει έλεγχο πολλαπλών καταστάσεων.

if grade >= 90:
    print("Άριστα")
elif grade >= 70:
    print("Καλά")
else:
    print("Ανεπαρκής")

4. Συνθήκες με and, or, not: Συνδυασμός πολλών λογικών ελέγχων.

if age > 18 and has_id:
    print("Πρόσβαση επιτρεπτή")

if temperature < 0 or is_raining:
    print("Πάρε ομπρέλα ή ντύσου καλά")

if not has_password:
    print("Απαιτείται κωδικός")

5. Σύντομη συνθήκη (Ternary Expression): Γράφεις if...else σε μία γραμμή.

print("Ενήλικας") if age >= 18 else print("Ανήλικος")

Προσοχή στην Εσοχή (Indentation): Η Python χρησιμοποιεί εσοχή αντί για αγκύλες. Αν δεν βάλεις σωστά την εσοχή, ο κώδικας δε θα εκτελείται.

if x > 0:
print("Θετικός")  # ← λάθος, λείπει εσοχή

Συμπέρασμα: Η if είναι το βασικό εργαλείο λογικής στον προγραμματισμό. Με τους συνδυασμούς elif, else, and, or, και not, μπορείς να προγραμματίζεις αποφάσεις όπως θα σκεφτόταν ένας άνθρωπος.
