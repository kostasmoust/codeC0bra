Variable Expressions

Μεταβλητές: Τιμές όπως αριθμοί και συμβολοσειρές (strings) λέγονται σταθερές. Οι αριθμοί γράφονται κανονικά (π.χ. print(123) → 123), ενώ οι συμβολοσειρές χρειάζονται εισαγωγικά (π.χ. print("Hello World") → Hello World). Μεταβλητές είναι ονοματισμένα μέρη της μνήμης που αποθηκεύουν δεδομένα (π.χ. x = 123, y = 13).

Τελεστές: Είναι σύμβολα που εκτελούν πράξεις σε τιμές και μεταβλητές:

1. Αριθμητικοί:

+  Πρόσθεση         → 5 + 3 = 8
-  Αφαίρεση         → 5 - 2 = 3
*  Πολλαπλασιασμός  → 4 * 2 = 8
/  Διαίρεση         → 8 / 2 = 4.0
// Ακέραια διαίρεση → 9 // 2 = 4
%  Υπόλοιπο         → 9 % 2 = 1
** Ύψωση σε δύναμη  → 2 ** 3 = 8

2. Τελεστές Σύγκρισης:

==  Ίσο              → 5 == 5 → True
!=  Όχι ίσο          → 5 != 3 → True
>   Μεγαλύτερο       → 7 > 3 → True
<   Μικρότερο        → 2 < 4 → True
>=  Μεγαλύτερο/Ίσο   → 5 >= 5 → True
<=  Μικρότερο/Ίσο    → 3 <= 6 → True

3. Λογικοί Τελεστές:

and → True and False → False
or  → True or False  → True
not → not True       → False

4. Τελεστές Ανάθεσης:

=   x = 5      (Απλή ανάθεση)
+=  x += 2     (x = x + 2)
-=  x -= 1     (x = x - 1)
*=  x *= 3     (x = x * 3)
/=  x /= 2     (x = x / 2)

Προτάσεις (Statements): Είναι εντολές που δίνουμε στο πρόγραμμα.

1. Ανάθεσης: x = 10, name = "Maria"

2. Εντολών: print("Γεια σου κόσμε!"), result = 5 + 3

3. Επανάληψης:

for i in range(5):
    print(i)

while x > 0:
    x -= 1

4. Απόφασης:

if x > 0:
    print("Θετικός αριθμός")
elif x == 0:
    print("Μηδέν")
else:
    print("Αρνητικός αριθμός")

5. Σύνθετες Προτάσεις:

def χαιρετισμος(ονομα):
    print("Γεια σου,", ονομα)

Προτάσεις Τερματισμού:

for i in range(10):
    if i == 5:
        break
    if i % 2 == 0:
        continue
    print(i)

pass  # Χρησιμοποιείται ως placeholder

Συμπέρασμα: Οι μεταβλητές, τελεστές και προτάσεις είναι τα βασικά εργαλεία κάθε προγράμματος σε Python. Μαθαίνοντάς τα, μπορείς να δημιουργήσεις από απλά έως πολύπλοκα προγράμματα.
