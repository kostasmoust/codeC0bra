Μεταβλητές
Τιμές όπως νούμερα ,γράμματα και συμβολοσειρές (strings) ονομάζονται σταθερές μεταβλητές ,οι αριθμοί γράφονται όπως ξέρουμε (print(123)>>>123) ενώ οι συμβολοσειρές χρειάζονται εισαγωγικά(") (print(‘Hello World’)>>>>Hello World)

Μεταβλητές είναι τα ονοματισμένα μέρη της μνήμης όπου μπορούμε να αποθηκεύσουμε δεδομένα και να τα ανακαλέσουμε με το όνομα τους οποιαδήποτε στιγμή ,τα ονόματα αυτά καθορίζονται από τον προγραμματιστή όπως και οι τιμες των μεταβλητων τις οποιες μπορει να αλλαξει και κατά την διαρκεια (x=123,y=13)

Τελεστές Στην Python οι τελεστές είναι σύμβολα ή λέξεις που χρησιμοποιούνται για να εκτελέσουν λειτουργίες σε τιμές και μεταβλητές.Είναι απαραίτητοι για την εκτέλεση υπολογισμών, συγκρίσεων και άλλωνβασικών λειτουργιών.Ας δούμε τους βασικούς τύπους τελεστών που θα συναντήσεις:
1. Αριθμητικοί Τελεστές (Arithmetic Operators)
Τελεστής| Περιγραφή | Παράδειγμα
+ | Πρόσθεση |5 + 3 → 8
- | Αφαίρεση |5 - 2 → 3
* | Πολλαπλασιασμός | 4 * 2 → 8
/ |Διαίρεση |8 / 2 → 4.0
// |Ακέραια διαίρεση |9 // 2 → 4
% |Υπόλοιπο |9 % 2 → 1
** |Ύψωση σε δύναμη | 2 ** 3 → 8
Αυτοί χρησιμοποιούνται για βασικές μαθηματικές πράξεις:

2. Τελεστές Σύγκρισης (Comparison Operators) Χρησιμοποιούνται για να συγκρίνουμε δύο τιμές και επιστρέφουν True ή False.
Τελεστής| Περιγραφή| Παράδειγμα
== | Ίσο με | 5 == 5 → True
!= | Όχι ίσο | 5 != 3 → True
> | Μεγαλύτερο | 7 > 3 → True
< | Μικρότερο | 2 < 4 → True
>= | Μεγαλύτερο ή ίσο| 5 >= 5 → True
<= | Μικρότερο ή ίσο | 3 <= 6 → True

3. Λογικοί Τελεστές (Logical Operators) Χρησιμοποιούνται σε λογικές εκφράσεις.
Τελεστής| Περιγραφή| Παράδειγμα
and |Και |(και οι δύο True) True and False → False
or |Ή |(τουλάχιστον ένα True) True or False → True
not |Άρνηση |(αντίστροφο του True) not True → False

4. Τελεστές Ανάθεσης (Assignment Operators)
Τελεστής |Παράδειγμα| Ισοδύναμο με
= |x = 5 |απλή ανάθεση
+=| x += 2| x = x + 2
-=| x -= 1| x = x - 1
*=| x *= 3| x = x * 3
/=| x /= 2| x = x / 2
Χρησιμοποιούνται για να δώσουμε τιμές σε μεταβλητές.

Οι τελεστές είναι θεμέλιο της Python και απαραίτητοι για να γράψεις ακόμα και το πιο απλό πρόγραμμα. Μόλις τους μάθεις, θα μπορείς να δημιουργείς δυναμικά προγράμματα, να συγκρίνεις τιμές και να παίρνεις αποφάσεις με βάση τις συνθήκες.


Προτάσεις
Στην Python, πρόταση (ή statement) είναι κάθε γραμμή κώδικα που λέει στον υπολογιστή να κάνει κάτι. Είναι οι «οδηγίες» που δίνουμε στο πρόγραμμα. Όταν εκτελούμε ένα πρόγραμμα, η Python διαβάζει τις προτάσεις μία προς μία και τις εκτελεί με τη σειρά.Υπάρχουν διάφορα είδη προτάσεων στην Python. Παρακάτω θα δούμε τις πιο βασικές:
1. Προτάσεις Ανάθεσης (Assignment Statements) Αναθέτουν μια τιμή σε μια μεταβλητή. (π.χ. δώσε στην μεταβλητή χ την τιμή δέκα: x = 10) (ή αποθήκευσε το όνομα «Maria» στη μεταβλητή name = "Maria") 2. Προτάσεις Εντολών (Expression Statements) Εκτελούν κάποιον υπολογισμό ή καλούν μια συνάρτηση. (π.χ. εκτύπωσε το μήνυμα «Γεια σου κόσμε!»: print("Γεια σου κόσμε!")) (ή υπολόγισε το άθροισμα του πέντε και του τρία και αποθήκευσέ το: result = 5 + 3) 3. Προτάσεις Επανάληψης (Loop Statements) Εκτελούν μια εντολή πολλές φορές. (π.χ. επανάλαβε πέντε φορές και τύπωσε κάθε αριθμό από το μηδέν έως το τέσσερα: for i in range(5): print(i)) (ή όσο το x είναι μεγαλύτερο από το μηδέν, μείωσέ το κατά ένα: while x > 0: x -= 1) 4. Προτάσεις Απόφασης (Conditional Statements) Επιτρέπουν στο πρόγραμμα να παίρνει αποφάσεις, ανάλογα με συνθήκες. (π.χ. αν το x είναι θετικό, τύπωσε "Θετικός αριθμός", αν είναι μηδέν, τύπωσε "Μηδέν", αλλιώς τύπωσε "Αρνητικός αριθμός": if x > 0: print("Θετικός αριθμός") elif x == 0: print("Μηδέν") else: print("Αρνητικός αριθμός")) 5. Σύνθετες Προτάσεις (Compound Statements) Αποτελούνται από πολλές γραμμές και συνήθως περιλαμβάνουν δομές όπως if, for, while, def και class. Περιέχουν άλλες προτάσεις στο εσωτερικό τους και χρειάζονται σωστή εσοχή (indentation). (π.χ. δήλωσε μια συνάρτηση με όνομα χαιρετισμος που τυπώνει μήνυμα: def χαιρετισμος(ονομα):   print("Γεια σου,", ονομα)) Προτάσεις Τερματισμού (Break, Continue, Pass) Χρησιμοποιούνται μέσα σε βρόχους ή ως placeholders. (π.χ. για κάθε i από το μηδέν έως το εννέα:   αν το i είναι ίσο με πέντε, σταμάτα τον βρόχο   αν το i είναι ζυγός αριθμός, προχώρα στον επόμενο   αλλιώς τύπωσε το i: for i in range (10):   if i == 5: break   if i % 2 == 0: continue   print(i)) (χρησιμοποίησε pass για να κρατήσεις θέση για μελλοντικό κώδικα: pass) Συμπέρασμα Οι προτάσεις είναι τα «δομικά υλικά» κάθε προγράμματος. Κάθε γραμμή κώδικα που γράφεις είναι μια πρόταση που λέει στην Python τι να κάνει. Μαθαίνοντας να τις χρησιμοποιείς σωστά, μπορείς να χτίσεις από απλά μέχρι και πολύπλοκα προγράμματα.
