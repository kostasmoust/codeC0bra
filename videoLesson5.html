Strings Ένα string είναι απλώς ένα κομμάτι κειμένου μέσα σε εισαγωγικά. Μπορεί να περιέχει λέξεις, φράσεις, αριθμούς, σύμβολα — οτιδήποτε. (π.χ. "Καλημέρα", '123', "Python είναι διασκεδαστική!") 🔹 2. Πρόσβαση σε χαρακτήρες Μπορείς να αναφερθείς σε συγκεκριμένους χαρακτήρες ενός string με δείκτες (index), ξεκινώντας από το 0. (π.χ. πάρε τον πρώτο χαρακτήρα του όνομα: όνομα = "Άννα" όνομα[0] → "Ά") (π.χ. πάρε τον τελευταίο χαρακτήρα: όνομα[-1] → "α") 🔹 3. Τεμαχισμός (Slicing) Μπορείς να πάρεις ένα υποσύνολο χαρακτήρων. (π.χ. πάρε τους πρώτους 3 χαρακτήρες του όνομα: όνομα[:3] → "Άνν") (π.χ. πάρε τους χαρακτήρες από τη θέση 1 μέχρι 3 (όχι συμπεριλαμβανομένου του 3): όνομα[1:3] → "νν") 🔹 4. Εντολές με string • len(string) → Μήκος του string (π.χ. len("Python") → 6) • string.upper() → Όλα κεφαλαία (π.χ. "python".upper() → "PYTHON") • string.lower() → Όλα μικρά (π.χ. "HELLO".lower() → "hello") • string.strip() → Αφαιρεί κενά από αρχή και τέλος (π.χ. " hello ".strip() → "hello") • string.replace("παλιό", "νέο") → Αντικαθιστά (π.χ. "Καληνύχτα Ελλάδα".replace("Ελλάδα", "Κόσμε") → "Καληνύχτα Κόσμε") • string.split() → Χωρίζει σε λίστα λέξεων (π.χ. "Καλημέρα σε όλους".split() → ["Καλημέρα", "σε", "όλους"]) 🔹 5. Ένωση strings Μπορείς να ενώσεις strings με το +. (π.χ. "Καλημέρα" + " " + "Κόσμε" → "Καλημέρα Κόσμε") 🔹 6. f-strings (String Formatting) Εύκολος τρόπος να βάλεις μεταβλητές μέσα σε κείμενο. (π.χ. όνομα = "Άννα" ηλικία = 25 f"Το όνομά μου είναι {όνομα} και είμαι {ηλικία} χρονών" → "Το όνομά μου είναι Άννα και είμαι 25 χρονών") 🔹 7. Χρήσιμες συναρτήσεις • in → Ελέγχει αν κάτι υπάρχει σε string (π.χ. "Python" in "Μαθαίνω Python" → True) • string.startswith("...") (π.χ. "Καλημέρα".startswith("Καλη") → True) • string.endswith("...") (π.χ. "καληνύχτα".endswith("ύχτα") → True) Συμπέρασμα Τα strings είναι παντού στην Python — από μηνύματα στην οθόνη μέχρι επεξεργασία δεδομένων. Είναι απλά, αλλά και πολύ ισχυρά όταν τα μάθεις καλά!
