Strings In Python

Strings: Ένα string είναι απλώς ένα κομμάτι κειμένου μέσα σε εισαγωγικά. Μπορεί να περιέχει λέξεις, φράσεις, αριθμούς, σύμβολα — οτιδήποτε.

"Καλημέρα", '123', "Python είναι διασκεδαστική!"

2. Πρόσβαση σε χαρακτήρες:

Χρησιμοποιούμε δείκτες (index) για να πάρουμε χαρακτήρες από string.

όνομα = "Άννα"
όνομα[0]     # → "Ά"
όνομα[-1]    # → "α"

3. Τεμαχισμός (Slicing):

Μπορούμε να πάρουμε υποσύνολα χαρακτήρων από ένα string.

όνομα[:3]     # → "Άνν"
όνομα[1:3]    # → "νν"

4. Εντολές με string:

len("Python")                   # → 6
"python".upper()               # → "PYTHON"
"HELLO".lower()                # → "hello"
" hello ".strip()              # → "hello"
"Καληνύχτα Ελλάδα".replace("Ελλάδα", "Κόσμε")  # → "Καληνύχτα Κόσμε"
"Καλημέρα σε όλους".split()   # → ["Καλημέρα", "σε", "όλους"]

5. Ένωση strings:

"Καλημέρα" + " " + "Κόσμε"    # → "Καλημέρα Κόσμε"

6. f-strings (String Formatting):

Εύκολος τρόπος να βάλεις μεταβλητές μέσα σε string.

όνομα = "Άννα"
ηλικία = 25
f"Το όνομά μου είναι {όνομα} και είμαι {ηλικία} χρονών"
# → "Το όνομά μου είναι Άννα και είμαι 25 χρονών"

7. Χρήσιμες συναρτήσεις:

"Python" in "Μαθαίνω Python"          # → True
"Καλημέρα".startswith("Καλη")         # → True
"καληνύχτα".endswith("ύχτα")          # → True

Συμπέρασμα: Τα strings είναι παντού στην Python — από μηνύματα στην οθόνη μέχρι επεξεργασία δεδομένων. Είναι απλά, αλλά και πολύ ισχυρά όταν τα μάθεις καλά!
